How to use Google Drive as Web Hosting (Free)

Hello everyone! Today, we're going to explore how to utilize Google Drive as our website hosting servers..

We must have a Gmail account. It's advisable to create a new Gmail account specifically for use as a web server. Next, we need to create a folder inside our Google Drive where we can store our website files.

Afterward, we upload all the website files to our Google Drive folder.Now, we require a shareable link to this folder, ensuring that the access level is set to "Anyone with the link."


Now, visit https://www.drv.tw/

Click on "Host on Google Drive," select your Gmail ID, then click on "Allow" so that "Drive To Web" can access your Google Drive.

Now, please wait a few seconds for the process to complete. Afterward, we'll be able to see the Admin Panel page. Now, you should be able to locate our website link section under Your Web Pages.

Connecting a Custom Domain to the Website

Let's go to the register Where we bought our domain , in my case main is Namecheap. Then go to domain manage on the DNS Setting remove the current name server Afterward go to Advanced DNS, add new cName Record, On the host fild type www and on the value field past website URL from the "Drive To Web", remove the HTTPS prefix & remove everythings after drv.tw

Set the TTL to 30 minutes and save changes

Now add another record, Select URL redriect on the host fild type @ & on the value field type http://domain.com (Your Domain Name) Now save changes

Done

Give some time maybe 1 Hours, Website will be live on your Domain
